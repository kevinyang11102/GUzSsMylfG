# 个人健康管理系统（前后端分离版） java769

## 技术栈

- 前端：Vue
- 后端：SpringBoot + Java
- 数据库：MySQL

## 项目概述

个人健康管理系统是基于前后端分离架构设计，旨在实现个人健康数据的数字化管理与智能分析。适用于医疗健康领域的学习实践、毕业设计开发或企业健康管理系统二次开发，提供全流程技术支持。

## 系统架构

### 架构模式

- B/S架构
- MVC分层设计

### 系统环境

- 支持Windows/macOS双系统
- 服务器端：SpringBoot 2.x（JDK 1.8）
- 数据库：MySQL 8.0

### 开发工具

- IDE：IDEA
- 包管理：Maven
- 前端：Vue CLI + Webpack

### 核心技术栈

- 后端：Java、SpringBoot、MyBatis、MySQL
- 前端：Vue、Element UI、Axios
- 部署：Nginx（可选）、Docker（可配置）

## 功能模块

### 健康档案管理

- 个人基础信息管理（身高/体重/过敏史等）
- 历史体检报告电子化存储与查询
- 家庭成员健康档案关联（可选扩展）

### 健康数据中心

- 多维度健康数据记录（血压/血糖/心率眠等）
- 数据可视化分析（趋势图表、异常值预警）
- 数据导入导出（Excel/CSV格式）

### 健康管理工具

- 用药管理：服药提醒、剂量管理、药品库存预警
- 目标跟踪：自定义健康目标（如减重/增肌）、进度可视化
- 运动管理：运动计划制定、卡路里消耗统计、运动轨迹记录

### 智能提醒服务

- 用药/体检时间提醒
- 数据异常预警（如血压超标自动提醒）
- 健康任务推送（每日饮水/休息提醒）

## 运行环境
